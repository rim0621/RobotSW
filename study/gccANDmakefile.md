# gcc
* c file =>(전처리기-#인것들) =>i file =>(어셈블리어로 변경)=> s file =>(어샘을 기계어로)=> o file =>(링커[파일들을 링크해준다.])=> a.out
* -v --save-temps 옵션을 사용해서 위 과정을 볼 수 있다.
* -o : 결과 파일명 설정
* -c : 오브젝트 파일(까지만) 생성
* -Wall : 모호한 문법에 대한 경고 메시지 출력
* -W : 모호한 코딩에 다한 부가적인 정보 제공
